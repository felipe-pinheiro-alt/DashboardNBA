# DashboardNBA
Este projeto é um dashboard interativo em Streamlit que analisa a evolução das bolas de 3 pontos na NBA, com foco nas temporadas de 2014–15 a 2023–24. O app coleta dados da API oficial da NBA via nba_api, salva em arquivos CSV locais e exibe visualizações e métricas comparando a liga com os times campeões.
